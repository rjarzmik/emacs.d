
Selecting articles for action:
 - key '#' to select one for processing
 - 'M' 'P' 'r' to mark the current region for processing
 - 'M' 'P' 'g' to unmark the current region for processing
 - 'M' 'P' 'i' to invert the articles marked for processing
 - B 'DEL' to delete marked articles

Mailing list specific commands:
 - C-c C-n p
   Post to the mailing list, if List-Post field exists.
 - C-c C-n o
   Send a message to the mailing list owner, if List-Owner field exists.
 - C-c C-n a
   Browse the mailing list archive, if List-Archive field exists.
